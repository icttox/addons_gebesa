Mexican Localization
====================

This module was created for the installation of the most important applications of the Mexican localization, which help the administration of accounting accounts in mexico.
These applications can be found on the menu of l10n_mx_base settings are listed the most important

* l10n_mx_base
* l10n_mx_payroll_base
* l10n_mx_accountinge
* l10n_mx_facturae_pac_finkok
* l10n_mx_facturae_pac_sf
* l10n_mx_diot_report
* account_voucher_tax

NOTE: Never should be deleted the MXN currency, This Could cause errors in the data and inconsistency
